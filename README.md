SpeckOrder
============

Order module for the ecommerce platform SpeckCommerce

Introduction
------------

Steps to get this module running:
- add `"speckcommerce/speck-order": "dev-master"` to `require` in your composer.json
- copy SpeckOrder/config/*.dist files to application autoload/config/ folder.
Edit as appropriate and remove .dist suffix
- create the tables necessary for this module, using SpeckOrder/data/schema.sql
